
# Weather Station App

This is a C# Console Application that demonstrates:

- Events and Delegates
- Multicast Delegates
- Anonymous Methods
- Event Handling in C#

## 🚀 Getting Started

### Prerequisites
- .NET SDK installed
- Visual Studio or VS Code

### 🔧 Setup

1. Build the project:
    ```sh
    dotnet build
    ```
2. Run the project:
    ```sh
    dotnet run
    ```
